# expense-tracker-python
Personal Expense Tracker built using Python

# Overview
The Personal Expense Tracker is a command-line application developed using Python that helps users record, manage, and analyze their daily expenses. The program allows users to store expense details such as date, category, amount, and description, enabling better financial tracking and organization.

# Features
- Add new expense records
- View all recorded expenses
- Calculate total spending based on a specific date
- Generate monthly spending reports
- Generate category-wise spending reports
- Delete expenses using multiple conditions (date, category, description)
- Clear all stored expense records

# Technologies Used
- Python
- File Handling
- Menu-driven program design

# How to Run the Project
1. Download the project file.
2. Open the file using Python IDLE or any Python environment.
3. Run the program.
4. Use the menu options to add, view, or analyze expenses.

# Learning Outcome
This project demonstrates practical implementation of Python concepts such as file handling, loops, conditional statements, and data processing in a real-world scenario.
